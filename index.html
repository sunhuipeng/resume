个人介绍(未在职)

孙会鹏 - 男 - 河南洛阳,很荣幸在 2018年7月毕业于郑州大学 计算机科学与技术(网络服务与工程)专业

码云主页: https://gitee.com/sunhuip
github主页: https://github.com/sunhuipeng


在大学期间积极的参加项目,扩展知识面,希望能在毕业后不再做纯粹的 码农,能和优秀的团队在一起战斗,总体来说我接触过并开发过以下类型的应用:

app

ios | android | react native

ios学习,至今没有做过正式项目,但积累了移动开发经验, android自学,有一个在项目的经验, react native自学,有一个项目经验

window

qt | c#

窗口类程序与app有一定的相似之处,使用 c++的 qt库开发过多个程序,在大三的时候,还自学 c#的UI开发框架做过小型商业项目

console

c++ | c#

c++大概有一年的开发经历,其中涉及到 第三方库的开发,整合, c#累计有一年半的开发经历,涉及到后台以及服务类程序的开发

web

c# | java | node.js

web后台开发使用过: asp.net, think.js , spring boot 前端开发使用过: vue.js, react.js

信息概览

在大学期间,通过大量的自学,实验室同学的耳濡目染以及大三的第一份实习工作,学习的痛苦到收获的喜悦,在成长过程中也参加了大量的项目 ,拥有了宝贵的经验. 在各个方面进行 预培养.

git版本控制

2017年7月正式入职 河南金聚科技有限公司后开始使用git方式对项目进行版本控制.之前使用的是实验室内部搭建的私 有svn方式进行版本控制, 所以7月之前的代码提交统计是不显示的.最近没有连续提交代码是因为处理毕业的事情,以及中间回家.

我最主要使用码云进行版本控制,码云允许添加免费的私有项目,我们这边把项目分摊在不同的负责人账号上,基本上满足团队开发需 求. 刚开始提交量比较多是因为转型后对git提交,冲突解决的熟悉过程,

码云和github的交集是维护CmsWing时的代码提交,我们公司在码云中 fork私有后并在基础上添加自己的业务,在使用中修复项目Bug或者公用代码后 重新 pull request给原项目,回馈开源社区,同时这也是我学习 node.js时的第一个项目.见证了我从一个node.js的菜鸟,到现在的 正常编写业务逻辑,实现需求.



c# 精选

为了完成 CaseSys,在大三开始接触c#编程语言.这也是我第一个web项目. 当时面临的问题有:1.项目需求梳理把控能力几乎没有;2. web的开发模式与 console有很大差别;3. c#, asp.net, javascript等语言的学习以及使用

DeviceServer是使用c#开发有图形化界面的设备服务器(主要用来研发,测试使用).这是"选线定位"第一二期的成品.在此项目之前 Socket, UI,以及 Console的开发基础.经前期架构设计后,实际编码中很流畅.

electr-server是"选线定位"第三期项目的一部分,对项目职责拆分,是一个纯服务型的 console服务.主要负责设备的接入,控制以及设备数据分析以及固化.与内部其他服务的通信方式为web-socket(那时候还不知道有 rpc)



c++ 精选

c++98是专业课,同时又学习了 c++11, c++14的新语法,新特性ps:(当时还误以为计算机语言一旦发明,就不会在发生变化). c++的新特性确实可以加快编程速度,简化程序结构.

反复拜读过 深入应用c++11代码优化与工程级应用 -- 祁宇编著,没有其他高级语言的经验,纯粹从 c++98到 c++11跨度是比较大的,我在反复拜读和实践中才逐渐体会到它的强大.

IdentificationDocuments和 IdentificationDocumentsTool是医疗单据识别一套程序,是给研究院开发的 dll模块



java 精选

java基础是专业课,因为我有, c++模板, 动态库挂载, c#泛型,反射的经验,后来也快速掌握了,java部分高级语法

platform是使用spring系列框架,开发的物联网后台程序,目前还未完成.



node.js 精选

node.js在之前的项目精力加成(写过jQuery插件,大致机制)下,学习到熟练相对来说比较容易.后来又学习了 think.js基于node.js的后台mvc框架,它整合了很多优质框架

node-moudle是自己写的非常小的rpc的demo集合.主要想到达, think.js作为api聚合层,然后调用多个后端 node.js为前端提供服务.

electr-data是"选线定位"第三期的数据层服务,主要对外提供api接口,消息推送,数据中转处理

CmsWing是一个基于 node.js的开源 cms项目,在码云中评为 GVP,我参与了项目的维护工作,系统后台管理首页有我的名字(是原作者加上去的).



vue.js | react.js 精选

vue.js, react.js以及 react native的使用,在有, node.js后端开发以及 app的基础上进行了快速的入门,在这期间对打包构建工具有了一定程度的认知.

vue-visa是给 新加坡的企业开发的用于在线办理签证,在线英语考试的项目

electr-admin是"选线定位"第三期的管理端页面,主要负责对数据的可视化,以及维护.

ElectrMobile是"选线定位"第三期的移动端应用,采用 react native混合式开发,给用户提供可以可视化终端.
